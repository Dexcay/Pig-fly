# How to install (Windows) :
1. You must have a 64-bit system
2. Select any [release](https://github.com/Dexcay/Pig-fly/releases) and download the "win64.zip" archive.
3. Unzip the "win64" folder and find "Хряк летит.exe" or "nw.exe" (depending on the version)
4. Run the exe file and enjoy the game!


# Minimum System Requirements :
- CPU 130 MHz
- RAM 120 MB
- VRAM 10 MB
- 100 MB free space
- 64-bit system
